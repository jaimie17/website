Team Projects on GitHub

COS1 107a: Introduction to Computer Security

I collaborated with a team member for a semester-long project on the ethics of network analyzers Nmap and Wireshark. We performed extensive research and safely tested out these network analyzers to learn and understand their implications on privacy and ethics.

COS1 103a: Fundamentals of Software Engineering

For this course, I collaborated with a team of 3 other students to create various web apps throughout the semester using JavaScript, Flask, SQL, Mongoose, and EJS.

GPT Grammarly

We created a "Grammarly" Flask web app using the ChatGPT API. Features include grammar editor, summarization or paraphrasing, synonym generator, poem generator, and Spanish translation based on input text.

Productivity App - Finance Tracker

We built a Finance Tracker using SQL, then recreated it in our Productivity web app using Express and Mongoose. Our finance tracker allows the user to add, delete, and edit transactions as well as sort them by different categories such as amount and date.

Productivity App - TextTrove

Building on our Productivity App, we added a TextTrove page similar to our GPT Grammarly which also uses the ChatGPT API to create poems, summaries, Spanish translations, and synonyms using input text.
